CIP::function

From cppreference.com
< cpp‎ | string
template<

   class ReturnT,

> class function;

This class wraps a function pointer which takes arguments (unsigned int argc, void * argv[]) with arguments to that function. Its purpose is to delay when code is run until if and when it needs to, so that input_handler may be provided code to excecutre later. It is intended that this class is only created through the functions fref and fptr (and other specialized creation functions for specific function pointers), but it is important that you understand how its constructor works in case you define functions of your own to use with this.

[edit] Notes

This class should never need to be created or accessed directly and should be created through fptr() or fref(), or other specialized creation functions.

Because all functions which are wrapped by this class must take their arguments as pointers, it may sometimes be necessary to pass pointers made with new to this function. This is the main reason why the constructor provides the ability to mark variables for destruction.
